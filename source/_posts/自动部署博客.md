不想用第三方的travis-ci实现自动部署博客，但是gitbub提供的钩子折腾了半天弄不好

就用刚学的部分shell命令写了一个,用git bash 运行

日后优化

```shell
#!/bin/bash

hexo generate &&    /

git add . && /

git commit -a -m "auto commit by ali at `date +%Y%m%d`" && /

git push origin master && /

mv -f public/* ../Ali1213.github.io && /

cd  ../Ali1213.github.io && /

git add . && /

git commit -a -m "auto commit by ali at `date +%Y%m%d`" && /

git push origin master
```
